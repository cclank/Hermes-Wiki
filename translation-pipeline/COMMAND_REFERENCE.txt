🔑 ENVIRONMENT VARIABLES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Required:
  CLAUDE_API_KEY          - Claude API key from console.anthropic.com

Optional:
  GITHUB_TOKEN            - GitHub token for private repos
  LOCAL_MODE              - true/false (default: false)
  LOCAL_STORAGE_PATH      - Local storage path (default: ./translations)
  GCP_PROJECT_ID          - GCP project ID (cloud mode)
  GCP_REGION              - GCP region (default: us-central1)
  GCS_BUCKET_NAME         - GCS bucket name
  MAX_WORKERS             - Parallel workers (default: 5)
  TRANSLATION_MODEL       - Claude model (default: claude-3-5-sonnet-20241022)
  MAX_TOKENS              - Max tokens per request (default: 4096)


💡 TIPS & TRICKS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Use local mode for testing:
   Set LOCAL_MODE=true in .env

2. Monitor progress with --monitor flag:
   python3 client_enhanced.py translate ... --monitor

3. Stop monitoring with Ctrl+C (task continues):
   Task keeps running in background

4. Check cached translations:
   Identical content returns instantly from cache

5. Customize terminology:
   Edit terminology_map.json to add your terms

6. Reduce costs:
   Use caching and local mode for development

7. Scale workers:
   Increase MAX_WORKERS for faster processing

8. Private repos:
   Set GITHUB_TOKEN in .env


🐛 TROUBLESHOOTING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Service won't start:
  • Check if port 8080 is in use: lsof -i :8080
  • Use different port: PORT=8081 python3 app_enhanced.py

Translation fails:
  • Verify Claude API key: echo $CLAUDE_API_KEY
  • Check service health: python3 client_enhanced.py health

Git clone fails:
  • For private repos, set GITHUB_TOKEN
  • Check repo exists and is accessible

Out of memory:
  • Reduce MAX_WORKERS in .env
  • Use cloud mode for large repos

Dependencies missing:
  • Run: pip install -r requirements.txt
  • Or: ./deploy.sh (auto-installs)
